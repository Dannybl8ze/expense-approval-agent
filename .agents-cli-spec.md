# Agent Spec: customer-support-agent

## Overview
A graph-based customer support workflow using ADK 2.0. The agent classifies if a user query is related to shipping or not, then routes the request accordingly.

## Example Use Cases
1. **Input:** "How do I track my package?" -> **Classifier:** shipping -> **FAQ Agent:** "You can track your package by entering your tracking number..."
2. **Input:** "What is the capital of France?" -> **Classifier:** unrelated -> **Decline Node:** "I'm sorry, but I can only help you with shipping-related queries."

## Tools Required
None.

## Constraints & Safety Rules
- The workflow must decline to answer queries not related to shipping.
- Do not make up shipping rates; refer to standard policy.

## Success Criteria
- Accurately classifies queries.
- Politely declines unrelated queries.
